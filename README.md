# DP-900
**DP-900: Microsoft Azure Data Fundamentals Study Guide & Hands-On Labs**

This repository provides learning resources, hands-on labs, and step-by-step walkthroughs to help learners prepare for the DP-900 Microsoft Azure Data Fundamentals certification.

The materials included here complement the DP-900 Study Guide and provide practical experience with core Microsoft Azure data services.

⭐ If this repository helps your learning journey, please consider starring the repository.

**About the DP-900 Certification**

The DP-900 Azure Data Fundamentals certification validates foundational knowledge of core data concepts and how they are implemented using Azure data services.

The exam focuses on:

1. Core data concepts
2. Relational data in Azure
3. Non-relational data in Azure
4. Analytics workloads in Azure

It is an ideal certification for beginners who want to understand modern cloud data platforms.

**What You Will Learn**

This repository covers the essential services used in Azure data solutions, including:

1. Azure SQL Database
2. Azure Cosmos DB
3. Azure Data Factory
4. Azure Synapse Analytics
5. Azure Data Lake Storage Gen2
6. Microsoft Fabric
7. Microsoft OneLake

The goal is to help learners understand how Azure data services work together to support modern analytics solutions.
